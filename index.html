工具 Tools

格式化与数据

JSON 格式化与校验 JSON

校验、美化、压缩 JSON，语法高亮、树形视图、错误定位、转义与统计信息。
JSON 格式互转 转换

JSON 转 YAML、TypeScript interface、Go struct 与 CSV，写配置定类型一步到位。
SQL 格式化 SQL

格式化或压缩 SQL，支持 MySQL、PostgreSQL 等常见方言与关键字风格。
YAML 转换与校验 YAML

YAML 与 JSON 双向转换、语法校验与错误定位，写配置排查缩进问题。
cURL 命令解析 HTTP

解析 cURL 命令为结构化请求，生成 fetch 与 Python requests 代码。

编码与字符

URL 编码与参数解析 URL

URL encode/decode、查询参数解析、参数对象互转，适合排查接口与分享链接。
Base64 编码转换 编码

UTF-8 文本 Base64 编码/解码，并支持 Hex、Unicode escape 与字节统计。
Unicode 字符检查器 Unicode

逐字符查看码点与编码，揪出零宽字符、同形字等看不见的坑。
HTML 实体编解码 HTML

HTML 特殊字符与命名/十进制/十六进制实体互转，附常用实体速查表。

文本处理

正则表达式测试 Regex

实时测试 JavaScript 正则，展示匹配高亮、捕获组和替换结果。
文本差异比较 Diff

左右文本行级对比，高亮新增、删除与未变内容，适合配置和日志片段。
文本命名与行处理 文本

camelCase / snake_case 等命名互转，行去重、排序、去空行与统计。
Markdown 预览 MD

Markdown 实时渲染预览，复制净化后 HTML 或下载完整页面。

时间与调度

时间转换 时间

Unix 时间与日期互转，秒/毫秒自动识别、批量转换、多格式输出与时区对照。
Cron 表达式解析 Cron

解析 5 字段 Cron，展示字段含义，并预览未来 10 次执行时间。
时区会议规划 时区

多城市时间对照与工作时段标注，跨时区约会议不再算错时间。

生成与凭证

随机 ID 生成器 随机

生成 UUID v4、随机密码、Token、数字串与批量自定义随机字符串。
Hash 计算器 摘要

计算文本或文件的 MD5 / SHA-1 / SHA-256 / SHA-512，支持拖拽文件。
JWT 解析器 Token

本地解码 JWT 的 Header 与 Payload，标注过期状态，可选 HS256 验签。
二维码生成器 外部

快速生成二维码，支持跳转到独立二维码生成服务，适合链接、文本和分享场景。
密钥对生成器 密钥

浏览器本地生成 RSA / ECDSA / Ed25519 密钥对，导出 PEM 与 JWK。

图表与图像

Mermaid 图表编辑器 图表

实时编辑渲染流程图、时序图等，模板库、缩放平移、分享链接与导出 PNG / SVG。
颜色转换与对比度 Color

HEX / RGB / HSL 互转，生成色阶，并检查前景与背景的 WCAG 对比度。
图片压缩与转换 图片

本地压缩并转换 WebP / JPEG / PNG，调质量看体积对比，不上传服务器。
SVG 优化与预览 SVG

SVG 本地优化压缩、预览与转 data URI，看体积对比。

速查参考

Emoji 符号库 Unicode

分类浏览与中英文搜索，收藏、最近使用、肤色切换与一键复制。
HTTP 状态码与 MIME 速查 速查

常用状态码含义与 MIME 类型对照，搜索过滤、点击复制。
chmod 权限计算器 Linux

Linux 文件权限三向换算：勾选、八进制与符号表示实时同步。

关于AES加解密

    
AES 是目前应用最广的对称加密算法，本工具采用 AES-256-GCM 认证加密模式：不仅加密内容，还能校验数据完整性，密文被篡改会直接解密失败。密钥由你输入的密码经 PBKDF2-SHA256 迭代 200000 轮派生，每次加密使用随机生成的 16 字节 salt 和 12 字节 IV，相同明文每次加密结果都不同。全部运算通过浏览器内置的 Web Crypto API 在本地完成，适合加密备忘、口令、配置片段等敏感文本。

    
常见问题

    
这个工具的加密强度如何？
采用 AES-256-GCM 认证加密，是目前主流的安全标准，兼具机密性与完整性校验；密钥由 PBKDF2-SHA256 迭代 200000 轮派生并配合随机 salt，显著提高暴力破解成本。实际安全性主要取决于密码强度，建议使用足够长的随机密码。

    
我的数据和密码会上传到服务器吗？
不会。加密、解密和密钥派生全部通过浏览器内置的 Web Crypto API 在本地完成，密码和文本内容都不会离开你的设备，断网状态下也能正常使用。

    
能解密其他工具加密的 AES 数据吗？
不能。本工具使用自定义的 zkaes1 自描述格式（前缀 + Base64 编码的 salt、IV 与密文），与其他工具的输出格式和密钥派生参数不同，只有本工具加密的内容之间可以互通解密。

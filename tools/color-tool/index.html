关于颜色转换

    
本工具支持 HEX（#2563eb）、RGB、HSL 三种颜色格式互转，输入任一格式即可得到全部写法；同时自动生成该颜色的 7 级深浅色阶（适合做主题色梯度），并按 WCAG 2.1 标准计算前景色与背景色的对比度，分别判定普通正文（4.5:1）和大字号（3:1）的 AA/AAA 达标情况，帮你在设计阶段就避免"文字看不清"的可访问性问题。

    
常见问题

    
WCAG 对比度多少才算达标？
普通正文要求至少 4.5:1（AA 级）、7:1 更佳（AAA 级）；18pt 以上大字或 14pt 粗体放宽到 3:1。灰字配白底很容易不达标，上线前值得检查一遍。

    
HEX、RGB、HSL 该用哪种？
三者表示同一颜色：HEX 最紧凑通用；RGB 便于程序计算；HSL 按色相/饱和度/亮度描述，调"同色系深浅"时最直观（只改 L 值）。CSS 中三种都可以用。

    
三位的 #fff 和六位的 #ffffff 一样吗？
一样。三位是缩写形式，每一位重复一次展开：#fff = #ffffff、#f80 = #ff8800。本工具两种写法都能识别。

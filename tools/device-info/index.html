关于设备信息面板

这是一个纯本地运行的设备信息查询工具：打开即自动读取屏幕分辨率、devicePixelRatio、可用区域、视口尺寸（innerWidth / visualViewport / clientWidth）、滚动条宽度、safe-area 安全区四边内边距、User-Agent、语言、硬件并发数、触屏与指针精度、深色模式与减弱动画偏好、网络类型与电池状态，并附浏览器能力速查表。所有数据一行一项，点击任意一行即可复制该值；顶部「复制全部信息」按钮可一键拼成多行文本，直接粘贴到 Bug 报告或工单里。旋转屏幕、缩放窗口时视口数据实时刷新并高亮变化的值，方便调试响应式布局。

常见问题

物理分辨率和 screen.width 为什么不一样？

screen.width 是 CSS 像素下的逻辑分辨率，物理分辨率 = 逻辑分辨率 × devicePixelRatio。例如 iPhone 显示 screen.width 为 390，DPR 为 3，物理分辨率就是 1170。做高清图适配、@2x/@3x 切图时以 DPR 为准。

innerWidth 和 clientWidth 为什么差几个像素？

window.innerWidth 包含垂直滚动条的宽度，而 document.documentElement.clientWidth 不包含，两者之差就是滚动条宽度（macOS 悬浮滚动条和多数手机浏览器为 0）。写媒体查询时浏览器用的是含滚动条的宽度，与 innerWidth 一致。

页面能读取这些信息，会泄露隐私吗？

本页读取的都是浏览器公开暴露给任何网页的常规属性（屏幕、视口、UA、语言等），不涉及权限授权，也正因如此它们常被用于浏览器指纹。本工具完全在本地运行，不发送任何网络请求，数据不会离开你的设备。

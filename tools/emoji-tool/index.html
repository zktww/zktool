关于Emoji大全

本工具收录全部 Unicode 标准 Emoji，按笑脸、人物、动物、食物等分类浏览，基于 Unicode CLDR 官方中文标注支持中文关键词搜索（比如搜"庆祝"或"猫"），点击即可复制到剪贴板。支持肤色切换、收藏常用表情和最近使用记录，也能查看每个 Emoji 的 Unicode 码点，方便在代码中使用转义形式。

常见问题

为什么有的 Emoji 在我的设备上显示为方框？

不同操作系统的 Emoji 字体版本不同：较新的 Emoji（新版 Unicode 加入的）在旧系统上没有对应字形，就会显示方框或问号。发送给别人时对方能否看到取决于对方设备。

Emoji 可以用在代码里吗？

可以，Emoji 是合法的 Unicode 字符，能用于字符串、注释甚至部分语言的变量名。注意一个 Emoji 可能占多个 UTF-16 编码单元，用 JavaScript 的 length 计算长度会"偏大"。

如何搜索想要的表情？

直接输入中文或英文关键词，如"爱心""火箭"或 rocket；搜索基于 Unicode CLDR 的官方中文名称和关键词标注，无匹配时会回退英文名称匹配。

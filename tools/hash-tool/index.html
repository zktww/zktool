关于MD5在线加密

哈希（Hash）把任意长度的数据变成固定长度的"指纹"，内容变一个字节指纹就完全不同，因此常用于校验文件完整性、生成缓存键和签名比对。本工具同时计算 MD5、SHA-1、SHA-256、SHA-512 四种摘要，支持直接输入文本或拖入文件（最大 256MB）；SHA 系列基于浏览器原生 Web Crypto，文件在本地读取计算，不会上传到任何服务器。

常见问题

如何校验下载的文件是否完整？

把下载的文件拖入本工具，对比计算出的 SHA-256（或官网提供的对应算法）值与官网公布的校验值是否一致；一致说明文件完整未被篡改。

MD5 还能用吗？

MD5 已被证实可以人为构造碰撞，不能再用于签名、证书等安全场景；但作为文件完整性校验、缓存键、去重标识等非对抗性用途仍然普遍且够用。安全场景请用 SHA-256 及以上。

同样的文本为什么算出的哈希和别人不一样？

检查三点：是否混入了首尾空格或换行、编码是否一致（本工具按 UTF-8 处理）、对方是否对内容做过其他预处理（如加盐）。哈希对每个字节都敏感。

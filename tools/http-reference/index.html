关于HTTP状态码大全

HTTP 状态码是排查接口和网页问题的第一线索：2xx 成功、3xx 重定向、4xx 客户端错误、5xx 服务端错误。本速查表收录 30 余个常用状态码，每条附中文含义和实际排查提示（比如 429 要看 Retry-After、502 通常是后端挂了）；另有常用 MIME 类型（Content-Type）与文件扩展名对照表。支持关键词搜索，点击条目直接复制。

常见问题

401 和 403 有什么区别？

401 Unauthorized 是"没登录或凭证无效"——缺 Token、Token 过期都返回它；403 Forbidden 是"登录了但没权限"——身份有效但该资源不允许你访问。前者去检查认证，后者去检查授权。

502 和 504 都是网关错误，怎么区分？

502 Bad Gateway：网关收到了上游的响应但无效，通常是后端进程崩溃或返回格式错误；504 Gateway Timeout：网关等上游等到超时，通常是后端处理太慢或卡死。一个是"回了但坏了"，一个是"根本没回"。

接口返回乱码和 Content-Type 有关吗？

很可能有关：响应头的 Content-Type 缺少 charset=utf-8 时，浏览器可能按错误编码解析。后端确保返回 application/json; charset=utf-8 这类完整声明即可。

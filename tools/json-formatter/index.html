关于JSON格式化

    
JSON 是接口传输和配置文件最常用的数据格式，但压缩后的 JSON 可读性差、手写的 JSON 容易有语法错误。本工具基于浏览器原生 JSON.parse 做严格校验，一键美化（可选 2/4 空格或 Tab 缩进）或压缩为单行，出错时定位到具体位置；所有数据在浏览器本地处理，不会上传服务器，可放心粘贴包含敏感信息的接口返回。

    
常见问题

    
JSON 校验失败提示 Unexpected token 怎么办？
常见原因：键名或字符串用了单引号（JSON 标准要求双引号）、最后一项多了尾逗号、值里有未转义的换行或引号。根据错误提示的位置逐一排查即可。

    
在线 JSON 格式化安全吗？会泄露数据吗？
本工具所有处理都在你的浏览器本地完成，数据不会发送到任何服务器，断网也能使用；适合处理包含 token、手机号等敏感信息的数据。

    
支持 JSON5 或注释吗？
不支持。本工具遵循标准 JSON 规范（RFC 8259），不接受注释、尾逗号、单引号等 JSON5 扩展语法，这也保证了校验结果与后端解析行为一致。

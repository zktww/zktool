关于Markdown在线预览

Markdown 是一种轻量级标记语言，用简单的符号（# 标题、- 列表、| 表格）就能写出结构化文档，广泛用于 README、博客、技术文档和笔记。本工具基于 marked 在浏览器本地把 md 文本实时渲染为 HTML，并内置经典蓝、清新绿、暖阳橙、紫罗兰、极简黑多套排版主题，支持自定义主题色、正文字号、Mac 风格代码块、首行缩进与外链转脚注（公众号不支持外部链接，自动转成文末参考链接），并提供全屏左右分栏编辑（编辑滚动预览跟随，Esc 退出）；所有样式都写成内联 style，点「复制到公众号」即以富文本形式进入剪贴板，粘贴到微信公众号、知乎、邮件等富文本编辑器后排版原样保留。渲染结果先经 DOMPurify 消毒（白名单过滤，移除脚本、事件属性与危险链接）再展示，也可复制 HTML 源码或下载为独立 .html 文件，全程不上传任何内容。

常见问题

支持哪些 Markdown 语法？

支持 CommonMark 标准语法（标题、段落、列表、引用、代码块、链接、图片、水平线等），以及 GFM 扩展的表格和删除线。数学公式、脚注等更小众的扩展语法暂不支持。

怎么把排版结果发到微信公众号？

选好主题和字号后点「复制到公众号」，然后到公众号后台编辑器里直接 Ctrl+V 粘贴即可，标题、引用、代码块等样式会原样保留（样式全部内联，不依赖外部 CSS）。同样适用于知乎、掘金等支持富文本粘贴的编辑器。

复制的 HTML 可以直接用吗？安全吗？

可以。输出的 HTML 已经 DOMPurify 白名单消毒：仅保留安全的标签与属性，移除脚本、事件属性和危险协议链接，可直接粘贴到邮件、CMS 或网页中。如果目标环境有更严格的安全要求，建议再过一遍服务端过滤。

我的文档内容会上传到服务器吗？

不会。渲染完全在浏览器本地完成，内容不经过任何网络请求；输入草稿仅保存在你自己浏览器的 localStorage 里，方便下次打开继续编辑，可随时点"清空"删除。

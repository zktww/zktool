关于Mermaid在线编辑器

Mermaid 是用文本语法画图的工具，写几行代码就能生成流程图、时序图、甘特图、状态图、ER 图等，特别适合放进 Markdown 文档和技术方案。本编辑器左侧写代码右侧实时渲染，内置十余种中文示例模板可直接改，支持缩放平移预览、URL 分享（图表内容编码在链接里）以及导出 SVG 和高清 PNG。

常见问题

Mermaid 和 Visio、draw.io 有什么区别？

Mermaid 用文本描述图形，改文字就能改图，适合放进 Git 做版本管理、嵌入 Markdown；Visio/draw.io 是拖拽式画图，自由度更高但不易维护。画技术架构和流程建议 Mermaid，画精确排版的图用后者。

GitHub / GitLab 支持 Mermaid 吗？

支持。在 Markdown 里使用 mermaid 代码块（```mermaid）即可直接渲染，本工具调好的代码可以原样粘贴过去。

导出的 PNG 不够清晰怎么办？

本工具导出 PNG 时按 2 倍分辨率渲染，一般场景足够清晰；对清晰度要求极高（如印刷）建议导出 SVG 矢量格式，任意缩放不失真。

关于UUID生成器

    
本工具基于浏览器加密安全的 Crypto API（而非 Math.random）生成随机内容：UUID v4、随机密码、URL-safe Token、数字串和自定义字符集字符串，支持一次批量生成最多 200 条。密码模式默认排除易混淆字符（如 0/O、1/l），并实时显示熵值强度评估；随机数使用拒绝采样消除取模偏差，每个字符的概率严格均等。

    
常见问题

    
UUID 会重复吗？
UUID v4 有 122 位随机熵，重复概率低到可以忽略——生成十亿个 UUID，碰撞概率也远小于万亿分之一。实际工程中可以放心当作全局唯一 ID 使用。

    
多长的密码才算安全？
看熵值：熵 = 长度 × log2(字符集大小)。60 bit 以下容易被暴力破解，80 bit 够日常使用，128 bit 以上非常安全。本工具会实时显示当前配置的熵值和强度评级。

    
这里生成的密码会被记录吗？
不会。生成过程完全在你的浏览器本地完成，不经过任何服务器，页面也不存储生成结果；刷新即消失。

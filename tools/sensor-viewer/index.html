关于传感器查看器

传感器查看器是一个在浏览器里直接测试手机运动传感器的在线工具：通过 DeviceOrientation 事件实时展示陀螺仪方向角 alpha/beta/gamma，并用 CSS 3D 手机模型直观还原设备姿态；通过 DeviceMotion 事件读取加速度计的含重力/不含重力 x/y/z 分量与旋转速率，配合条形图和合加速度模长展示；指南针面板在 iOS 上优先使用系统罗盘朝向。所有数据仅在本地读取与渲染，不会上传到任何服务器，适合排查设备传感器故障、学习 Web 传感器 API 或做体感交互调试。

常见问题

为什么点击按钮后没有任何数据？

台式机和大部分笔记本没有陀螺仪与加速度计，桌面浏览器虽然有 API 但不会产生事件，请扫码用手机打开本页。另外传感器 API 要求 HTTPS 安全上下文，通过 file:// 或 http 打开页面时无法读取。

iOS 上提示权限被拒绝怎么办？

iOS 13 及以上要求用户手势内授权，若之前点了"不允许"，Safari 会记住该选择。请到 设置 → Safari → 动作与方向访问 打开开关后刷新本页，再点击开始按钮重新授权。

alpha、beta、gamma 分别代表什么？

alpha 是设备绕垂直 Z 轴的偏航角（0~360°），beta 是绕横向 X 轴的俯仰角（-180~180°，前后倾斜），gamma 是绕纵向 Y 轴的横滚角（-90~90°，左右翻转）。指南针朝向在 iOS 上取 webkitCompassHeading，其他平台以 360-alpha 近似，可能存在偏差。

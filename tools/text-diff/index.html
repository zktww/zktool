Text diff

文本差异比较

左右输入两段文本，进行行级差异对比；适合配置、日志片段、接口返回内容的快速比较。

输入

差异

关于文本对比

本工具对左右两段文本做行级差异对比（基于 LCS 最长公共子序列算法），新增行绿色、删除行红色高亮，一眼看出两个版本改了什么。适合对比配置文件修改前后、两次接口返回、日志片段和文档版本；针对大文本做了优化（自动剥离相同的开头结尾），万行文本也能秒出结果，全程本地处理不上传。

常见问题

和 git diff 有什么区别？

算法同源（都是行级 diff），但本工具即贴即比，不需要文件进 Git；适合对比剪贴板里的临时内容。需要字符级精细对比或三方合并时，还是用专业 diff 工具更合适。

为什么明明"看起来一样"却显示有差异？

大概率是不可见差异：行尾空格、Tab 与空格混用、全角空格或零宽字符。可以把文本粘到本站的 Unicode 字符检查器里查看隐藏字符。

支持多大的文本？

常规文本（几千行内）实时对比；两侧差异区特别大时会自动降级为整段对比并提示，避免浏览器卡死。日常配置、日志片段的规模完全没有问题。
